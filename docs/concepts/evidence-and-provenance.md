# Evidence And Provenance

State must be traceable.

## Evidence References

Evidence references point to source material, not duplicated blobs.

Examples:

- meeting note id
- email id
- chat thread id
- task id
- document id
- metric snapshot id
- prior journal entry id

## Interpretation Versus Fact

The model may record interpretations, such as "campaign narrative feels diffuse."

External facts require evidence, such as "client approved the proposal."

## Rule

The system can persist low-confidence interpretations, but it must label
uncertainty and avoid converting interpretation into fact without evidence.
